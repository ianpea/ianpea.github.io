Admin Portal

(Part of Integrated Logistics System)

After the creation of a booking in the Customer Portal, the booking can be processed by the admins in the Admin Portal. Features:
  • Manage bookings
    • Including dividing customer bookings into smaller tasks
    • Edit the booking as per customer requirements if needed
  • Monitor Tasks Generated
    • Check if a forecasted item has entered/exit the warehouse.
    • Monitor the status of a truck job delivery and its location on the map (Google Maps API)
  • View the FULL warehouse configurations
    • Check each customer's product stock, balance, expected inbound, expected outbound, shortfall, etc
    • Admins can see a visualised warehouse with access to every customer's stock
    • Filter through items by name, SKU or even the owner of each item.
  • Manage customers
    • Create new customer user and edit their profile
  • Many more

View/monitor/edit the Booking

Here, the admins are able to access the customer's booking and edit its details
  • Edit details
  • Edit locations
  • Edit items
  • Upload relevant documents for reference (e.g. receipts, barcodes, excels)
  • Create tasks for the booking
*Note that edits can only be made when the booking has not been processed yet (i.e. pending task creation)
