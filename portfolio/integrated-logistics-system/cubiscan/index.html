Cubiscan Gantry API

(Part of Integrated Logistics System)

The Cubiscan is able to receive & update measurements of dimensions, weight from the Cubiscan Scanner to our database.

Receiving from Cubiscan

  1. Anytime an item is being loaded into the warehouse, it must go through the gantry.
  2. The gantry will weight and measure the dimensions of the pallet/parcel/box, and send an api call to our system which will register the values of that recorded item/product.
Cubiscan Tab in Admin Portal
Here you can see:
  • Measurements of package
  • Weight of package including pallet
  • Assigned customer of the package, if no customer (owner), the header will be yellow to highlight that it should be resolved quickly.
Cubiscan Tab in Admin Portal
For adhoc packages
  • Normally packages that come in are anticipated by the system. So for adhoc packages that are not anticipated, the admin will need to rectify error and point it to an existing item/customer.
